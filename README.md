Description :

```
The main objective of this project is to develop an application where user will be able to book appointments, they will be able to interact with the doctors using chat, audio and video call functionality. There will be no subscription instead they have to fund the wallet. It will be prepaid system and charges will deduct for audio/video call or chat.
Admin will be able to approve/ reject the doctor’s profile before they are published on the platform. Admin will manage the components of the application.

```
Key Features :

```
- Patient will be able to register into the platform by filling the medical details.
- Add funds to wallet
- Patient will be able to add the medical history details which will help taking the doctor’s guidance.
- Schedule Appointment
- Audio / Video call
- Patients will be able to interact with the doctor using the chat functionality
- Doctor will be able to register into the platform 
- Manage Wallet
- Doctor will be able to view the bookings made by the patients.
- Doctor will be able to accept / reject the bookings 
```
